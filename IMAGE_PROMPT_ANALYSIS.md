# Image Prompt Analysis & Update

## Analisa Prompt Sebelumnya vs Contoh yang Diberikan

### ❌ Masalah Prompt Lama:
1. **Terlalu generik** - Satu prompt untuk semua layout
2. **Fokus pada text** - Meminta text legible di gambar (AI image generator buruk untuk ini)
3. **Terlalu banyak detail typography** - Instruksi font, banner, border yang kompleks
4. **Tidak spesifik per layout** - Tidak ada instruksi visual yang berbeda untuk setiap layout

### ✅ Prompt Baru (Berdasarkan Contoh):
1. **Spesifik per layout** - Setiap layout punya instruksi visual yang berbeda
2. **Fokus pada visual** - Menekankan komposisi, struktur, dan elemen visual
3. **Instruksi yang jelas** - Step-by-step apa yang harus digambar
4. **Minimal text** - Tidak memaksa AI menulis text di gambar

## Perbandingan Detail

### Prompt Lama:
```
Create a VERTICAL EDUCATIONAL INFOGRAPHIC POSTER about GOLD PROSPECTING.

TEXT CONTENT TO INCLUDE (Must be legible):
HEADLINE: "..."
SUBTITLE: "..."
LIST POINTS: ...

MANDATORY ART DIRECTION:
- HEADER BANNER: Add a weathered, off-white/beige ribbon banner...
- TYPOGRAPHY: Main title: Heavy condensed sans-serif...
- FRAME/BORDER: Use full-bleed composition...
[Banyak detail teknis yang sulit diikuti AI]
```

**Masalah:**
- Meminta AI menulis text (AI buruk untuk ini)
- Terlalu banyak detail teknis
- Tidak ada perbedaan antar layout

### Prompt Baru:
```
Create a VERTICAL EDUCATIONAL INFOGRAPHIC about GOLD PROSPECTING.

TOPIC: [Topic Name]
KEY INFORMATION TO VISUALIZE: [Key points]
LAYOUT STYLE: [Specific Layout]

VISUAL EXECUTION:
[Layout-specific instructions - berbeda untuk setiap layout]

MANDATORY REQUIREMENTS:
- Format: Vertical 9:16
- Color Scheme: Earth tones with gold accents
- NO ABSTRACT ART. NO CARTOONS.
```

**Keunggulan:**
- Fokus pada visual composition
- Instruksi spesifik per layout
- Lebih mudah diikuti AI
- Hasil lebih konsisten

## 10 Layout dengan Instruksi Spesifik

### 1. CROSS-SECTION CUTAWAY
```
Create a realistic cross-section illustration showing underground layers.
Display surface at top, soil/gravel layers, bedrock at bottom.
Show gold deposits trapped in crevices.
Style: Educational textbook diagram.
```

### 2. VISUAL CHECKLIST (SPLIT SCREEN)
```
Create a split-screen comparison with vertical divider.
Left side shows one condition, right side shows contrasting condition.
Each side clearly labeled.
Style: Realistic field guide photography.
```

### 3. STEP-BY-STEP PROCESS
```
Create vertical flow diagram with numbered steps (1,2,3,4).
Use arrows indicating progression.
Each step shows clear stage of process.
Style: Illustrative with outdoor aesthetic.
```

### 4. FIELD SIGNS GRID
```
Create 2x2 or 3x2 grid with distinct close-up images.
Each cell clearly separated with borders.
Macro-style, highly detailed, realistic.
Focus on texture and detail.
```

### 5. THE GOLDEN PATH
```
Create top-down aerial view or map-style illustration.
Use arrows to mark flow direction.
Highlight zones with golden glow or markers.
Style: Strategic diagram or treasure map.
```

### 6. THE MAGNIFYING GLASS
```
Show surface with magnifying glass overlay.
Inside lens: highly magnified detailed view.
Sharp focus inside, blurred outside.
Style: Scientific discovery emphasis.
```

### 7. BEFORE & AFTER
```
Split landscape view - same location, two states.
Top half 'BEFORE', bottom half 'AFTER'.
Show clear changes between states.
Style: Realistic comparative photography.
```

### 8. THE GEOLOGIST'S NOTEBOOK
```
Page from field notebook appearance.
Hand-sketched drawings with handwritten annotations.
Arrows, labels, coffee stains, dirt smudges.
Background: Aged paper texture.
```

### 9. 3D BLOCK DIAGRAM
```
3D isometric block diagram - cutaway earth section.
Surface features on top, underground in cross-section.
Show geological features from deep to surface.
Style: Clean educational 3D technical illustration.
```

### 10. THE PROSPECTOR'S MAP
```
Vintage topographic map with contour lines.
Rivers in blue, hand-drawn X marks in red.
Important locations marked.
Style: Aged paper, treasure map aesthetic.
```
